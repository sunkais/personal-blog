## 2. 框架和构建命令

- **框架**：Astro 6.0.5（纯静态，无前端框架）
- **构建命令**：`npm run build`（→ `astro build`）
- **开发命令**：`npm run dev`（→ `astro dev`）
- **预览命令**：`npm run preview`（→ `astro preview`）
- **输出目录**：`dist/`（11 个页面 + RSS + sitemap）
- **数学渲染**：remark-math + rehype-katex + 自定义分隔符插件（服务端渲染）

## 9. 数学公式渲染修复

- **问题1**：首页公式使用 `<sup>` 手写 HTML → 改为 `<Formula>` 组件渲染
- **问题2**：`\{` 和 `\}` 在 Astro 模板中被当作表达式 → 使用 `{'{'}` 和 `{'}'}` 转义
- **问题3**：HTML 属性中 `\f`（`\frac`）被 JS 字符串解析为换页符 → 改为 `\\frac`
- **新建** `src/components/Formula.astro`：服务端调用 `katex.renderToString()` 渲染，设置 `throwOnError: false`
- 博客文章中的公式通过 remark/rehype 管道正常渲染（markdown 中 `\[...\]` 和 `\(...\)` 语法）
- KaTeX CSS 从 CDN 加载（`cdn.jsdelivr.net/npm/katex@0.16.9/dist/katex.min.css`）
