## 2. 框架和构建命令

- **框架**：Astro 6.0.5（纯静态，无前端框架）

- **开发命令**：`npm run dev` → `astro dev`
- **预览命令**：`npm run preview` → `astro preview`
- **输出目录**：`dist/`（11 个页面 + RSS + sitemap）
- **数学渲染**：remark-math + rehype-katex + 自定义 `\[...\]` / `\(...\)` 分隔符插件（完全服务端渲染）

## 9. 数学公式渲染本轮验证

**配置栈（无变化，已验证正常工作）：**
```
astro.config.mjs:
  remarkPlugins: [remarkMath, remarkTexDelimiters]
  rehypePlugins: [rehypeTexDelimiters, rehypeKatex]
```

- `remark-math` + 自定义 `remarkTexDelimiters`：将 `$...$` / `$$...$$` / `\[...\]` / `\(...\)` 识别为数学节点
- `rehypeTexDelimiters` + `rehype-katex`：将数学节点通过 KaTeX 渲染为 HTML
- `Formula.astro`：使用 `katex.renderToString()` 在非 markdown 场景下服务端渲染公式
- KaTeX CSS v0.16.9 通过 CDN 加载
