# 初始化修改说明

## 1. 当前项目结构简述

当前项目是一个 Astro 静态个人站点，主要目录如下：

```text
public/                 静态资源、字体、favicon
src/components/         Header、Footer、BaseHead 等通用组件
src/content/blog/       Markdown/MDX 笔记内容集合
src/layouts/            文章页布局
src/pages/              文件路由：首页、About、Blog/Notes、RSS
src/plugins/            Markdown/HTML 处理插件
src/styles/global.css   全站样式
astro.config.mjs        Astro 配置
package.json            npm 脚本和依赖
```

## 2. 框架和构建命令

项目使用 Astro 6，保持静态站结构，没有引入后端、登录、评论或用户系统。

常用命令：

```sh
npm run dev
npm run build
npm run preview
```

生产构建命令为 `npm run build`。
